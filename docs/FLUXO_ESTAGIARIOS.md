# Fluxo para testar estagiários de todos os departamentos

- Para cada departamento, crie o perfil com prefixo:
  - FINANCEIRO_ESTAGIARIO, TI_ESTAGIARIO, VENDAS_ESTAGIARIO, COMPRAS_ESTAGIARIO, ESTOQUE_ESTAGIARIO, MARKETING_ESTAGIARIO, PROJETOS_ESTAGIARIO, JURIDICO_ESTAGIARIO.
- Use o Assistente de Perfil com:
  - Módulo único do departamento.
  - Nível Básico.
  - “Gerar Recomendações” e “Aplicar ao Formulário”.
- Cadastre um usuário por departamento e vincule o perfil correspondente.
- Faça login com cada usuário e valide o sidebar:
  - O módulo correspondente deve aparecer.
  - Itens gerenciais não aparecem para estagiários.
