# Documentos Necessários (Processo Jurídico)

## Objetivo
- Padronizar anexos exigidos e recomendados para processos.

## Lista de Documentos
- Obrigatórios:
  - Petição inicial (PDF)
  - Documentos de identificação (RG/CPF ou CNPJ e contrato social)
  - Procuração
- Recomendados:
  - Contratos relevantes
  - Provas (fotos, vídeos, laudos)
  - Comprovantes (pagamentos, correspondências)

## Requisitos de Arquivos
- Formatos: PDF, DOCX, JPG, PNG, MP4 (para vídeos, se aceito)
- Tamanho máximo por arquivo: 25 MB
- Nomenclatura:
  - Use nomes claros e datados quando possível
  - Ex.: `peticao_inicial_2026-01-01.pdf`, `contrato_servicos_cliente_xx.pdf`, `foto_incidente_01.jpg`

## Estrutura Sugerida de Pastas (no repositório de documentos)
```
/processos/<numero_processo>/
  /peticao_inicial/
  /contratos/
  /provas/
  /comprovantes/
  /outros/
```

## Orientações
- Garanta legibilidade e qualidade das imagens.
- Evite duplicidades; mantenha versão mais atual.
- Registre metadados (data, autor, tipo) quando o sistema suportar.
