Central de Ajuda

Guia completo de acessos por módulo e subnível do sidebar.

Como Solicitar Acessos

Solicite acessos informando o módulo e os subníveis necessários. Use solicitações com escopo mínimo para evitar concessões excessivas.

  • Acesse: Serviços → Solicitações → Nova Solicitação
  • No campo módulos, selecione os módulos necessários
  • No campo subníveis, informe os authorities dos itens desejados (ex.: MENU_CLIENTES_LISTAR)
  • Após aprovação, o usuário é criado e recebe um perfil custom com apenas as permissões aprovadas

Exemplo Prático: Clientes

Para acessar apenas Clientes → Geral → Listar:

  • Módulo: Clientes
  • Authorities solicitadas: MENU_CLIENTES, MENU_CLIENTES_LISTAR
  • Resultado: o usuário verá o módulo Clientes e apenas o item “Listar” dentro de Geral

Solicitação de Acessos por Módulo
